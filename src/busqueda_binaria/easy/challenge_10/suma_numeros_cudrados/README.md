### _Suma de números cuadrados_

_Dado un entero no negativo c, decide si hay dos enteros a y b tales que a2 + b2 = c._

#### _Ejemplo 1_

```typescript
// Entrada:
let c = 5;
// Salida: verdadero
// Explicación: 1 * 1 + 2 * 2 = 5
```

#### _Ejemplo 2_

```typescript
// Entrada:
let c = 3;
// Salida: false
```

# _Restricciones_

- `0 <= c <= 2^31 - 1 [0 ; (2^31 - 1)]`
